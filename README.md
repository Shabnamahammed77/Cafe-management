# Cafe Management System (Python)

## Description
This is a console-based Cafe Management System developed using Python.
The project allows cafe owners to manage menu items, take customer orders,
and generate bills.

## Features
- Add menu items
- Prevent duplicate menu entries
- Display menu
- Take customer orders
- Generate total bill

## Technologies Used
- Python
- Object-Oriented Programming (OOP)

## How to Run
1. Clone the repository
2. Run main.py
3. Follow menu options

## Future Enhancements
- Store menu in database
- Add login system
